# julianhigman.com

Personal site for Julian Higman, senior Rails consultant.

Plain HTML and CSS — no framework, no build step. Hosted on GitHub Pages at [julianhigman.com](https://julianhigman.com).

## Local development

```sh
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Files

- `index.html` — the whole site
- `css/main.css` — all styles
- `CNAME` — custom domain for GitHub Pages
- `.nojekyll` — disables Jekyll processing on GitHub Pages
